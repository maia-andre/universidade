# Métodos de Delegação

Delegar não é simplesmente 'se livrar' do trabalho ('delargar'), mas sim um processo planejado que envolve:

1. **Escolha da Pessoa:** Avaliar competências, carga de trabalho e perfil para o desafio.
2. **Definição de Parâmetros:** Deixar claro qual é o resultado esperado, os limites de autoridade e a data de entrega.
3. **Apoio e Monitoramento:** Estar disponível para tirar dúvidas, sem exercer microgerenciamento.
4. **Avaliação Conjunta:** Analisar o resultado entregue como oportunidade de aprendizado.
